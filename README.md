#  Seller Dashboard – Fullstack E-commerce Platform

Fullstack project using NestJS + React.js to manage a seller's product catalog with media, SEO, stock, variants, and reviews.

---

## Swagger API Docs

Une fois le backend lancé :
http://localhost:3000/api



## Project Structure

- `product-catalog/`: NestJS API
- `frontend/`: React.js Dashboard
- `docs/`: Technical decisions and Swagger
- `postman_collection.json`: Postman requests
- `.env`: Sample env variables

## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```



# Backend
cd product-catalog
npm run start:dev

# Frontend
cd ../frontend
npm start

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
